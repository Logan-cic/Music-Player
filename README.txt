Da última apresentação até aqui foram adicionados: barra de volume, barra de progressão de música interativa,
possibilidade de adicionar diretórios específicos para ler músicas, opção de aleatório, opção de repetição de músicas
na playlist, repetição de uma única música e barra de menu.

Vale lembrar que não há nenhuma pasta selecionada por padrão para ler músicas, para isso é necessário abrir a primeira
opção da barra de menu, selecionar a opção de Adicionar Diretório e selecionar a pasta desejada. Recomenda-se que se
adicione os dois diretórios de músicas contidos na pasta Song_Files, já que são um espaço amostral com todos os
formatos em mp3 e com os metadados devidamente preenchidos.

Para abrir o reprodutor, deve-se executar o script main.py, contido na pasta GUI 1. Ao abrir o reprodutor, depois de
selecionar as pastas de música, basta apertar no botão play ou no botão de next para reproduzir as músicas.
